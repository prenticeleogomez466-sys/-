# L17 深化 · 顶级分析师/模型精髓 vs 本模型(吸取了什么、缺什么)

> "吸取最精华"的核心是诚实对照顶级看差距:统计核心已吸收~80%,真缺口是免费拿不到的事件级/球员级数据。精髓吸收 ≠ 超市场。

## 一、顶级方法 → 本模型吸收状态

| 顶级精髓 | 代表 | 本模型状态 |
|---|---|---|
| 攻防双系数+τ低分修正 | Dixon-Coles(1997) | ✅ 核心引擎 dixon-coles-engine |
| Elo 作 Poisson 协变量+中立场+Monte-Carlo | Groll / Hvattum-Arntzen | ✅ world-cup-priors + champion-sim |
| 球队评级系统 | 538 SPI / pi-ratings / Massey-Colley | ✅ 多评级(Elo/pi/Massey/Colley),pi主客场国际赛价值小 |
| 收盘线=信息聚合基准 | Betting Odds Rating(PLOS) | ✅ 市场为基准、blend塌缩到市场;实证打不过收盘线 |
| isotonic/校准 | Guo et al. | ✅ isotonic双map(模型/市场路径) |
| CLV/速度=真edge | sharp betting | ✅ capture-closing/clv-tracker;实证edge在速度非命中 |
| xG(射门质量) | Opta/StatsBomb/Understat | 🟡 仅训练期shot-regressed;实时xG免费只FBref(反爬) |

## 二、真缺口(免费拿不到→本模型够不到顶级的部分)
- **事件级 xG / xG chains / 射门坐标**:需 Opta/StatsBomb 事件流,免费仅 FBref(强反爬),实时拿不到。
- **球员级贡献 / 实时伤停阵容**:仅 FPL(英超)稳定,五大联赛+国家队薄。这是顶级模型真正领先处。
- **战术 matchup / 阵型 / pressing / set-piece 专项**:需事件级,免费无。
- 诚实:这些缺口**不是工程能修的,是免费数据天花板**(遵 free-only)。

## 三、下一刀 edge 在哪(诚实)
- **不在 1X2**:胜负平市场已高效(收盘线~54-55%,模型打平已是好结果)。
- **在市场低效角落**:低分/比分/大小球的**赔率薄小联赛**(瑞超/挪超/日职等市场定价不充分处)——L0"覆盖优先于精度"正为此。
- **在速度**:抢收盘前价(CLV),非提命中率。
- **在私有/早到信息**:实时伤停阵容比市场早(免费难,偶尔领先)。

## 四、诚实总结
统计核心已吸收~80%、与学界主流同构;再堆统计模块边际≈0(已用 judge panel+回测证)。真正差顶级的是**免费拿不到的事件级/球员级数据**,以及**速度**。命中率天花板 ~50-55% 不因"吸收更多精髓"突破——精髓吸收提升的是分析深度与诚实度,不是胜率。
