# L13 深化 · 验证回测纪律 + 本模型实证基线

> 怎么验证才可信:leak-safe 是底线,净增益是唯一采信标准。下面是可照抄的回测纪律 + 本模型实测过的真实基线数(都来自 walk-forward,可复现)。

## 一、回测纪律(可照抄)
1. **leak-safe walk-forward**:每个测试日只用**严格早于该日**的赛果拟合 DC/装配 context;绝不让终赛比分/收盘赔率/赛后xG 进训练。
2. **三臂对比**:纯DC / +信号融合 / +校准,量化每层真实边际贡献(而非只看最终数)。
3. **单信号消融(ablation)**:A=基线 vs A+某信号LR,看 Brier/LogLoss 是否真降——这是判"某模块有没有用"的唯一方法。
4. **诚实拒绝判据**:模型 RPS/LogLoss 须打过**开盘线**才算有独立判别力;打不过=与公开数据先验一致,不接主路径。
5. **改完必回测**:任何参数/模块改动,跑净增益回测,无净增益(超噪声~0.002)不改。

## 二、指标怎么读
| 指标 | 含义 | 用途 |
|---|---|---|
| 命中率 | argmax 命中 | 直观但粗(忽略概率质量) |
| Brier | 概率与0/1距离平方 | 概率校准+准确 |
| RPS(有序) | 累积分布差(home<draw<away) | 1X2 首选(惩罚方向错得离谱) |
| LogLoss | -log(P[真实]) | 重罚自信的错(校准敏感) |

## 三、本模型实证基线(leak-safe,可复现)
| 玩法/臂 | 命中率 | 备注 |
|---|---|---|
| 胜平负 market 市场赔率 | ~54.8% | 基准(含全公开信息,极难超) |
| 胜平负 blend 赔率+DC | ~54.2% | 打平市场已是好结果 |
| 胜平负 纯DC(俱乐部) | ~51.1% | 无赔率上限 |
| 胜平负 纯Elo(国际赛) | 50.5% | 判别力68.1% |
| 比分 DC-τ | ~12.5% | 物理上限12-15% |
| 半全场 泊松卷积 | ~31.4% | 上限28-35% |
| 大小球 | 打平市场 | 信心分层有用 |
| 双选(均势) | 40%→72% | 均势单选转双选 |

## 四、防泄漏 checklist(呼应 L19)
- [ ] 训练数据是否严格早于测试日?(用未来Elo/收盘赔率=泄漏)
- [ ] backcast 高命中率?(>60% 世界杯/>57% 联赛多是泄漏)
- [ ] 对照基线对吗?(单路模型该 vs 市场+vs 同类模型,别只挑有利对照)
- [ ] agent/judge 报的增益,自己 leak-safe 复现过吗?(judge① state-space+3.6pp 复现为持平)
- [ ] 描述统计差异当成预测增益了吗?(阶段λ/halfRatio 三连负教训)

**总则**:回测的目的不是证明模型好,是**诚实地找出它哪里不好**。命中率天花板 ~50-55%,任何宣称突破的先查泄漏。
