# 去AI味扫描表(定稿前对照 + 跑 deslop_scan.py)

> 提炼自网文去AI味实践,按"玄幻热血系统爽流"裁剪。**定稿前必须跑** `python scripts/deslop_scan.py <文件>`;退出码 2=重度必改。本表是人工复扫依据。

## 最毒句式(命中即改,Gate A 第一遍必扫)
| 毒级 | 句式 | 错误例 | 修法 |
|---|---|---|---|
| ★★★★★ | "不是A,而是B" / "不是A,不是B,而是C" | 他不是冷漠,而是绝望 | 直接写 B,或换更自然表达 |
| ★★★★ | ",带着……" 万能状语 | 他笑了一下,带着一丝不易察觉的嘲讽 | 拆短句或换具体动作 |
| ★★★★ | "声音不大,却带着一种……的力量" | 她声音不大,却带着不容置疑的力量 | 直接写声音特征或动作 |
| ★★★★ | "他/她知道/意识到/感到……" | 他知道这一切都来不及了 | 用行为展示认知 |
| ★★★ | "仿佛/犹如/宛若……一般" | 仿佛能穿透一切一般 | 删掉或白描 |
| ★★★ | "眼中闪过一丝…/嘴角勾起一抹…" | 眼中闪过一丝悲伤 | 他垂下眼 / 他笑了一下,没到眼底 |
| ★★★ | "心中涌起一股…/心头一震" | 心中涌起一股暖流 | 用身体反应 |
| ★★ | 章末空泛预告 "他不知道的是……" | 他不知道的是,更大的风暴即将来临 | 换具体钩子物件/事件收束 |

## 一级禁用词(出现即替换)
- 情态:仿佛 犹如 宛若 如同 一丝 一抹 些许 几分 隐约
- 动作:深吸一口气 缓缓 不禁 微微 轻轻 淡淡
- 表情:眼中闪过 嘴角勾起 眉头微皱 瞳孔微缩
- 心理:心中一动 心头一震 心下了然 心中暗道 不由得 不由自主 情不自禁
- 判断:不容置疑 不易察觉 显而易见 毫无疑问
- 形容:坚定 闪烁着光芒 狡黠 深邃 凛冽 冰冷

## 二级(高频才处理)
- 突然 好像 瞬间(角色口语/真实突发/时间压缩时可保留)
- 总结句:"他终于明白…""这才意识到…""此刻,他…""原来…"——删
- 排比:连续3句以上同结构 → 留最强一条
- 升华:"这一刻…""这就是…"——删

## 比喻(默认全删)
带 像/如/仿佛/犹如/宛若 的比喻是AI强信号。改为直接描述/动词/结果/事实。
"脸色惨白得像这漫天的雪" → "脸色惨白"。例外:生活化、角色身份能说出口的比喻可留(如赌徒说"这把稳赔")。

## 替换策略速查
| 原文类型 | 替换 | 例 |
|---|---|---|
| 抽象情绪词 | 具体动作 | 紧张 → 手在抖 |
| "感到XX" | 外化表现 | 感到愤怒 → 攥紧拳头 |
| 形容词堆砌 | 白描 | 美丽动人的笑容 → 她笑了 |
| 书面表达 | 口语化 | 不容置疑 → 就是 |
| 解释性描写 | 留白 | 他因为害怕而… → 他退后一步 |
| 多余修饰 | 删 | 白色的药片→药片;手里那截链子→链子 |

## 三遍去AI法
1. **预防**:写之前就把上面禁用词记牢,落笔即避。
2. **扫描**:`deslop_scan.py` 跑命中清单 + 本表人工复扫。
3. **改写**:逐条按替换策略改;★★★★★ 一处都不留。改完再跑一遍脚本确认退出码≤1。
