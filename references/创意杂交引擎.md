# 创意杂交引擎 —— 新意来自远域相撞

> 核心命题:**爆款创意 = 两个相隔很远的领域相撞**,不是在玄幻内部换金手指。50部里评分最高的 `法务流`(合同×修仙)、`现实bug猎手`(程序员×异能)就是证明。这份给"系统化撞新点子"的方法 + 反套路自检。

## 一、杂交公式
```
新创意 = 题材簇 × 远域元素 × 机制家族
远域元素 = 现实职业 / 学科 / 行业黑话 / 冷知识 / 社会议题 / 神话民俗
新意强度 ∝ 三者之间的"领域距离"
```
- 题材簇:见 `题材矩阵.md`
- 机制家族:见 `金手指机制图谱.md`
- **远域元素**:把现实世界一个具体职业/学科的底层逻辑,搬进金手指——这是新意的来源。

## 二、远域元素清单(金手指的"现实骨架")
- **职业**:程序员(找bug/提权/补丁)、律师(合同/举证/钻空子)、会计(资产负债/做账)、医生(诊断/手术)、厨师(火候/配方)、刑警(侦讯/侧写)、产品经理(需求/迭代)、保险精算(概率/赔付)、考古(还原/断代)、谈判专家、债务催收、殡葬师、锁匠、调香师、配音、电竞教练
- **学科**:博弈论、热力学(熵)、概率统计、经济学(供需/套利)、语言学(命名/能指)、生态学(食物链)、拓扑、密码学、运筹学、神经科学(记忆/认知)
- **行业逻辑**:股市(做空/杠杆/CLV)、拍卖、典当、租赁、外卖众包、直播打赏、信用评分、版本运营、KPI考核
- **冷知识/民俗**:二十四节气、阴阳五行的真实算法、古代刑名、堪舆、傩戏、各地葬俗、星象、行会规矩
- **社会议题**:35岁职场淘汰、内卷、996、彩礼、学区房、容貌焦虑、网暴、算法茧房——做成代入痛点 + 翻盘爽

## 四、反套路差异化自检(防撞车/防套路化)
- [ ] 这个创意,搜得到三本以上同款吗?(搜得到=红海,要么换机制要么加远域元素)
- [ ] 金手指的"现实骨架"够不够具体?(越具体越新;"系统"太泛,"会计做账"够硬)
- [ ] 十章细纲是不是每章同一个节拍重复?(50部高频毛病:扫面板→看穿→碾压×10)
- [ ] 反派是不是降智配合主角?(给反派至少一次真正威胁/反杀机会)
- [ ] 配角会不会都一个腔调?(给主要配角各自的说话指纹,见下)

## 五、角色 voice 指纹(防"所有人一个腔调"的深层AI腔)
立项时给每个主要角色一行说话指纹,写对白时贴着走:
```
{角色}:口头禅/句长/用词层级(粗俗?文雅?行话?)/不会说的话/潜台词习惯
例:赌徒主角——爱用赔率黑话("这把稳赔")、短句、从不示弱、嘴上认怂心里算账
```
所有角色对白都一个味 = 比"仿佛宛如"更深的AI腔,deslop_scan 抓不到,靠这一步预防。

## 六、批量生成接法
Workflow 选题阶段:架构师不再只按"金手指机制分桶",改成 **题材簇 × 远域元素** 双维分桶,每桶限定一个题材+一类职业/学科骨架,天然拉开领域距离——比纯机制分桶更不撞车、题材更丰富。可配合 `scripts/idea_forge.py` 先批量撞候选,再人工/评委筛。
