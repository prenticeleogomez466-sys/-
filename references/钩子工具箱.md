# 钩子工具箱 + 爽点引擎

> 写每章开头/结尾时查。先按情绪选钩子类型,再套模板。连续两章不用同一种钩子。

## 章尾钩子选型(按本章情绪)
| 本章情绪 | 推荐钩子 |
|---|---|
| 悬念/未知 | 突然揭示 / 神秘物品 / 留白 |
| 紧迫/危机 | 紧急危机 / 倒计时 / 未完成动作 |
| 反转/真相 | 身份反转 / 隐藏含义 / 回声 |
| 情感/关系 | 两难抉择 / 承诺威胁 / 情感转折 |
| 系统文 | 系统激活 / 选择型 |
| 氛围/日常 | 意象钩子 / 离奇消失 / 信息差 |

## 章尾钩子 13 式
| # | 名称 | 示例 |
|---|---|---|
| 1 | 突然揭示 | 信上的日期,是他死后第七天。 |
| 2 | 紧急危机 | 裂缝在扩大,灵石还差三块。 |
| 3 | 未完成动作 | 他刚伸手——"别动。"身后传来一个声音。 |
| 4 | 身份反转 | 她叫林小月。但档案上写的是:林小月,已故。 |
| 5 | 两难抉择 | 签了这份文件,公司保住了,但他得进去。 |
| 6 | 神秘物品 | 包裹里是一把钥匙,附了张纸条:"你欠我的。" |
| 7 | 倒计时 | 医生说还有三个月。那是两个月前的事了。 |
| 8 | 承诺/威胁 | 今晚十二点之前,我会告诉所有人你做了什么。 |
| 9 | 离奇消失 | 手铐还在,人没了。整个房间只有一个门,门没开过。 |
| 10 | 隐藏含义 | 他说:"你和妹妹真像。"——她是独生女。 |
| 11 | 意象钩子 | 枯了一个月的茉莉,突然冒出花苞。 |
| 12 | 回声钩子 | 章尾呼应开头,关键细节变了(她的手攥得更紧了)。 |
| 13 | 留白钩子 | 他看了信。脸色变了。什么也没说,把信叠好放进口袋。 |

## 章首钩子 7 式(前100字必须有,禁风景/天气开头)
| # | 名称 | 示例 |
|---|---|---|
| 1 | 悬念对话 | "你确定要这么做?""确定。""那你就别后悔。" |
| 2 | 闪前碎片 | 后来他才知道,那个电话改变了所有事。但此刻他还一无所知。 |
| 3 | 倒计时开局 | 距离合同到期还有 72 小时。 |
| 4 | 神秘独白 | 那天如果我没回头,一切会不会不一样。 |
| 5 | 反差场景 | 一边婚礼香槟塔,另一边医院走廊的灯在闪。 |
| 6 | 未完成动作 | 他刚把钥匙插进锁孔—— |
| 7 | 意象预示 | 天边烧得通红。她抬头看一眼,继续走。 |

## 章末实战模板(系统流常用)
**系统激活型**
```
就在{主角}{最低谷}之际。{一个感官描写}。叮!【{系统名},正式启动。】
```
**悬念预知型**(装逼前)
```
{主角冷静看着对方嚣张}。{短句:笑吧/慢慢来}。{预告反转,不透露细节}。
```
**截断型**
```
{一个小动作}。{剧烈反应}。{震惊/疑问——本章结束,不给答案}。
```
**选择型**
```
【选择一:{诱人}】奖励{极诱} 代价{隐含风险}
【选择二:{另选}】奖励{另诱} 代价{另风险}  ——不给结果。
```

## 爽点引擎
**爽点核心公式**:爽点 = 两个逻辑的冲突点 ——
- 大众逻辑:对手不可战胜 → 恐惧/绝望
- 主角逻辑:正好手痒 / 轻描淡写
- **落差越大,越爽。**

**爽点三要素**:吃瓜群众吹高对手逼格→震惊完成打脸升华;对手嚣张到读者恨;主角铺垫时低调、释放时干脆。

**装逼打脸节奏(5章循环)**:人设1章 → 铺垫2章 → 打脸1章 → 善后1章。
- 前置小无敌:打脸前先把道具/实力准备好,否则嘲讽变虐主。
- 铺垫>打脸:铺垫到位,主角站着不说话都能装逼。
- 打脸对象要有分量:路人震惊不叫爽,有头有脸的人震惊才叫爽。

## 写完自检
- [ ] 章首前100字有钩子(非风景/天气)
- [ ] 章尾有让人想翻页的东西,约100字,点到即止
- [ ] 钩子强度匹配章节阶段(第1章必须强钩)
- [ ] 连续两章没用同一种钩子
- [ ] 这一拍的爽点,前面铺垫到位了吗(看点=爽点一一对应)
