# 结构级判官 —— 抓 deslop_scan 抓不到的深层AI腔 + 逐章留存

> deslop_scan.py 只抓**词级**AI腔(仿佛/不是A而是B)。**结构级**AI腔(匀速流水句、降智反派、套路桥段、角色同腔、升华收尾)正则抓不到,必须上 LLM-judge(一个 subagent 读稿打分)。本件给判官协议 + 打分 schema。定稿/批量终检时跑。

## 一、深层AI腔 8 查(judge 逐条过)
| # | 病灶 | 判据 | 修法 |
|---|---|---|---|
| 1 | 匀速流水句 | 连续多句长度相近、主谓宾排队推进("他走进。他看着。他开口。") | 长短句交错,砸节奏。**style_stats 句长方差<6 是硬信号** |
| 2 | 降智反派 | 反派全程配合主角、不利用主角弱点、该反杀时发呆 | 给反派至少一次真威胁/差点得手 |
| 3 | 套路桥段重复 | 每章同一节拍("扫面板→看穿→碾压"×N)、可预判 | 每2-3章换冲突形态/打脸对象层级 |
| 4 | 角色同腔 | 所有角色一个说话方式,换名字看不出谁在说 | 上 `创意杂交引擎.md` 的角色voice指纹 |
| 5 | 升华式收尾 | 章末/段末用"这一刻…他终于明白…"拔高总结 | 换具体钩子物件/动作收束 |
| 6 | 信息密度均匀 | 通篇一个语速,没有"快推-慢炖"的疏密 | 关键场景放慢给感官,过场加速 |
| 7 | 似曾相识(撞名作) | 设定/腔调像辰东/番茄某书的复刻 | 从人物身份长出专属声音,别套模板 |
| 8 | 解释过度 | 凡事都交代因果动机,不留白 | 克制不说破,给读者推 |

## 二、judge 打分 schema(批量时给 subagent)
```json
{ "idx": 1, "flowSentence": "无/轻/重", "dumbVillain": "无/轻/重",
  "tropeRepeat": "无/轻/重", "sameVoice": "无/轻/重", "preachyEnding": "无/轻/重",
  "structAiScore": 1-10, "worstHit": "最重的一条结构AI腔", "fix": "怎么改" }
```
判定:`structAiScore < 7` 或任一项=重 → 打回结构级改写。

## 三、逐章留存(弃书风险)judge —— 别等读者用脚投票
一次性评级不够,要逐章问"读者会不会点下一章"。

**逐章自检清单**
- [ ] 本章前100字有钩子(非风景/天气/设定交代)
- [ ] 本章有一个明确爽点或悬念推进(没有=水章=弃书高发)
- [ ] 章末钩勾人,且和上一章不同类型
- [ ] 本章信息增量够(读者学到/猜到了新东西)
- [ ] 没有连续2章主角不动作/只铺垫

**留存曲线 judge schema**
```json
{ "chapter": 1, "retentionRisk": "低/中/高",
  "dropReason": "若高,读者会在这章弃书的原因",
  "hookStrength": 1-10, "isWaterChapter": true/false }
```
**红线**:黄金前三章任一章 `retentionRisk=高` → 必改;中段连续2章 `isWaterChapter` → 合并或加冲突。

## 四、数值/代价一致性核对(连载验证实测补的关键一刀)
"文件当记忆"能防人设漂移/伏笔遗漏,但**挡不住源头数值逻辑错误**——记忆机制只保证"记得",不保证"两个数本来就该相等"。2026-06-03 连载验证(《阴司账》5章)实测暴露三类,审校必须人工核一遍**关键数字**:
- **数字断言核对**:正文说"A 和 B 同步/吻合/一起到期",就去核 A、B 的具体数值是否真相等。验证里"认骨辨命剩6天"被断言与"倒计时7天14小时"同步归零,差了一天半——这种正则和judge都易放过,要专门核。修法:改数对齐,或保留差值改成悬念("为什么不是六天")。
- **代价红线核对**:设定表写死的代价(如"书面存证=坏账+1"),正文/状态档执行时有没有被**悄悄归零**?违背立项红线"代价不被偷偷忽略"。每用一次金手指,核对价是否真兑现。
- **阈值/条件核对**:设定"坏账≥20才出现反噬体",正文别在坏账=3时就让它现身;先兆与正式现身要在状态档分层标注。
> 核对清单:① 凡"同步/吻合/正好"的数字断言,逐一验算 ② 金手指每次使用,代价有没有真扣 ③ 触发阈值与正文表现是否对齐。这三条 deslop/style_stats/结构judge 都抓不到,是**连续性审计的必查项**。

## 五、跟脚本的分工
- 词级AI腔 → `deslop_scan.py`(确定性,退出码2必改)
- 节奏匀速 → `style_stats.py` 句长方差<6(确定性信号)
- 结构级(降智/套路/同腔/留存)→ 本判官,LLM subagent 读稿(正则做不到)
三层都过,才算真去AI + 抗弃书。
