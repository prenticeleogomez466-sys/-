# 黄金配方 + 文风锚定

## 一、黄金配方(选题先用这个,L1)
开局势能 =(金手指当量 × 反差幅度)÷ 释放速度
一句话开局 =【极致处境】+【金手指给的明确翻盘预期】

四件套咬合在**同一个困境**上:
- **金手指(翻盘当量)**:即时型(签到/抽奖/吞噬,当章见效)/ 成长型(系统/血脉,长线)/ 信息型(重生/预知/读心,信息差碾压)
- **身份(代入插口)**:低位(废物/赘婿/打工人,翻身爽)/ 隐藏高位(扮猪大佬,反差爽)
- **初始处境(憋屈势能)**:被退婚/发配/全家针对/欠债/绝症倒计时/当众社死——越具体越好
- **反差(点火器)**:把落差写到读者眼前,要有一个看低主角的对象来反衬

**配方铁律(配前排雷)**:
1. 金手指越强,初始处境越惨(反比),否则没坑可跳、爽不起来。
2. 金手指必须**直接**撬动眼前的局(欠高利贷却抽中"剑术精通"=够不着,废)。
3. 反差必须写到眼前(有围观/对比的人),不能只在设定里。
4. 别又是"废物少爷+退婚+吞噬"第N遍——换零件换组合,反自我模板。批量时**预分配不同金手指机制**,避开已写家族(签到/吞噬/回收/轮回/词条/命交易/奖池/夺光环/进化)。

## 二、文风锚定(给了对标书时,L5)——最强去AI味
"贴着一本真书的可量化特征写" = 比任何规则都有效的去AI味。

**步骤**
1. 拿到对标书原文(哪怕几章),跑 `python scripts/style_stats.py <对标书.txt>`,记下:短句占比 / 平均句长 / 标点密度 / 对白占比。
2. 人工读3段,提取**潜台词方式**(对白怎么藏话)、**情绪交替节奏**(紧张↔松弛怎么切)、**标点习惯**。
3. 写一个 `文风.md`:
```markdown
# {对标书} 文风锚
- 句长:短句X% / 中句Y% / 长句Z% / 平均N字   ← style_stats 实测
- 标点密度:M%   对白占比:K%
- 对白潜台词:{怎么不把话说满}
- 情绪节奏:{几段紧张接一段喘息}
- 可借鉴套路 Top3:{...}
- 禁忌:{这本绝不会出现的腔调}
```
4. 写作时贴着这些数字走;写完用 `style_stats.py` 测自己的稿,对不上就调(短句不够→拆句;对白太少→加带潜台词的对白)。

**爽流默认参考线(没有对标书时)**:短句<15字 ≥55%;平均句长 ≤18;对白占比 ≥25%;节奏要"砸"。

## 三、爽点引擎速查
(详见 `钩子工具箱.md`)爽点=大众逻辑(对手不可战胜)与主角逻辑(手痒)的落差;装逼打脸5章循环:人设1+铺垫2+打脸1+善后1;前置小无敌、铺垫>打脸、打脸对象要有分量。
